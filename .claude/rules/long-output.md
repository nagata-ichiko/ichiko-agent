# 長文出力はファイルへ

チャット応答の長文出力は、出力トークン上限による切断とコンテキスト消費の両方を招く。
一定量を超える出力はファイルに書き出し、チャットは要約に留める。

## ルール

- **生成文書はチャットに全文を出さない** — 設計書・レポート・長い説明はファイルに Write し、チャットには「要約3〜5行 + ファイルパス」のみ返す
- **レビュー結果が長くなる場合はファイルへ** — 指摘が多い場合は `logs/reviews/` 等に書き出し、チャットには深刻度別の件数サマリと修正必須項目のみ列挙する
- **PR コメント・Issue 本文の長文は --body-file で投稿** — チャットに全文を表示してから転記するのではなく、ファイルに書いて `gh pr comment --body-file` / `gh issue comment --body-file` を使う
- **サブエージェントへの指示も同様** — 長大な調査結果・レビュー結果を返させる場合はファイルパスに書き出させ、最終応答はサマリのみ返させる

## 目安

- 100行を超える見込みの出力はファイル行き
- 「全文を見せて」と明示されたときのみチャットに展開する
